# **4.4.1\. Setting Schedule Erasure for Erasing Traces**

 You can schedule a task to scan and erase internet traces, browser traces, system traces and application traces that are created while operating the system.

#### **Steps to set schedule Erasure for erasing traces:**

1. Run ****Stellar File Eraser** .
